# 🪶 Prompt de génération — Logo Kalamundi

*Créé le 15 juillet 2026. À coller dans un générateur d'image (Ideogram, GPT Image, Midjourney, Flux). Conseil : générer le **symbole SEUL** (sans texte — les IA écrivent mal), puis ajouter le mot « Kalamundi » en police **Fraunces** dans un éditeur.*

---

## 🎯 Prompt principal (symbole seul — recommandé)

```
Minimalist flat vector logo mark for "Kalamundi", an African digital book
and reading platform whose motto is "La Plume du Monde" (The World's Pen).

Core concept: a single elegant feather/quill that also reads as a leaf —
organic, African, literary. The feather doubles as a page or a leaf, symbol
of writing, nature and storytelling. Clean central spine with a few soft barbs.

Style: modern, geometric, minimal line-and-fill icon, confident and timeless.
Must stay legible and recognizable at very small sizes (favicon, app icon).
Enclosed in a soft rounded shape (circle or squircle) as an app badge.

Colors: deep forest green (#1B4332) as the base, warm gold (#D4A017 / #E9C46A)
for the feather. Warm cream background (#FBF8F2). Two colors only.

Flat design, no gradients, no 3D, no drop shadows, no photorealism, no clutter,
no text, no letters, single centered mark, generous negative space.
Vector, crisp edges, high contrast. On a plain background.
```

## 🎨 Rappels de marque (à respecter)

| Élément | Valeur |
|---|---|
| Nom | **Kalamundi** |
| Slogan | La Plume du Monde · Kalamu ya Dunia · The World's Pen |
| Symbole ancre | **la plume** (qui évoque feuille / page / monde) |
| Vert principal | `#1B4332` (foncé) · `#2D6A4F` (moyen) |
| Or accent | `#D4A017` · `#E9C46A` (clair) |
| Fond | crème chaud `#FBF8F2` |
| Style | flat, minimal, 2 couleurs, lisible en favicon |
| À éviter | texte dans le symbole, dégradés, 3D, ombres, surcharge, emoji 📚 |

## ✅ Après génération
1. Choisir la piste, vectoriser (SVG) si le rendu est en PNG.
2. Décliner : couleur / monochrome vert / inversé (or sur vert) / favicon.
3. Régénérer les icônes PWA via `generate-icons.mjs`.
4. Remplacer l'emoji `📚` dans `navbar__logo` / `footer__logo` sur **toutes** les pages.
